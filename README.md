# Minesweeper, in Java
One of the three games I made in 2021 while learning OOP using a textbook. <br/> <br/>
As I only knew Java back then, this uses Java awt to render all of the graphics and buttons. Also, the game was not designed to run on other computers, so
all the graphics were saved locally, on a computer I can no longer use. This will cause an error when attempting to run the program. <br/><br/>

When the local images problem is fixed, though, the game provides most of your standard minesweeper experience. Toggle between flag mode and click mode, reveal tiles,
and even a game over message. The only feature missing is the guarantee that the first click is always a 0.
